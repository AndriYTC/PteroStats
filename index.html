Customize Embed

You can change the embed in the config.json file!

Easy Setup

All you need to do is create apikeys and server at your pterodactyl panel, add the server id to index.html, and fill in the required informations in the config.json file.

How it works?

PteroStats checks pterodactyl server(s) resource api with axios to get server resource, if the api didn't reply that mean the node of the server is getting wings/daemon down and mark the node as offline
